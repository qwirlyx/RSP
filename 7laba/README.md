## Описание
Контейнеризированное Flask-приложение ToDo List с:
- Хранением задач в PostgreSQL
- Кэшированием списка задач в Redis
- Автоматическим перезапуском контейнеров
- Управлением через Docker Compose

## Структура папки
- `app.py` — основное приложение
- `Dockerfile` — сборка образа
- `docker-compose.yml` — запуск трёх сервисов (web, db, redis)
- `requirements.txt` — зависимости

## Как запустить
```bash
# Перейди в папку с проектом
cd путь_к_папке

# Сборка и запуск всех контейнеров
docker-compose up --build
